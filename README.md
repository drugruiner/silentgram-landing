# SilentGram Landing

Premium static landing page for SilentGram built with Vite, React, TypeScript, Tailwind CSS and Framer Motion.

## Development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

The static production files are generated in `dist/` and can be deployed to GitHub Pages, Nginx, Apache or any VPS static hosting setup.

## GitHub Pages

This project includes a GitHub Actions workflow that builds and publishes `dist/` to GitHub Pages on every push to `main`.

Repository Pages URL after deployment:

```txt
https://drugruiner.github.io/silentgram-landing/
```

If the repository name changes, update `base` in `vite.config.ts`.
